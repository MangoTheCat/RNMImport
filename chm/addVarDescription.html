addVarDescription(RNMImport)R Documentation

Adds a variable description

Description

Adds a new variable and appropriate characteristics to a global data frame for future reference, e.g. one may want to add a variable DOSAGEMG.

Usage

addVarDescription(varName, varLabel, varType, varFormat = NULL)

Arguments

varName The variable name
varLabel The variable label
varType The variable type
varFormat The variable format. NULL by default

Value

None
